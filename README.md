# CAN Bus Development Board

A general-purpose breakout and development board featuring the **MCP2562FDT** CAN transceiver, designed for testing and system integration.

## Key Features
* **Transceiver:** MCP2562FDT-E/SN (supports CAN FD up to 5 Mbps).
* **Power Rails:** Dual-rail design ($V_{DD} = 5\text{V}$ for bus, $V_{IO} = 3.3\text{V}$ for logic compatibility).
* **Protection:** On-board TVS diodes for ESD protection and space for a Common-Mode Choke.
* **Termination:** Includes optional split termination ($2 \times 60\,\Omega$ with a center-tapped capacitor to GND).
